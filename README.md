# School Portal - Forms & Admissions System

A web-based school portal that allows students to apply for admission,
view fees, and gives administrators a dashboard to manage applications.

🔗 **Live Demo:** https://weezy1234556.github.io/school-portal-frms

## Features
- Student admission application form
- Fees information page
- Admin login and dashboard
- Clean, responsive interface

## Built With
- HTML5
- CSS3
- JavaScript

## Pages
| Page | Description |
|------|-------------|
| index.html | Homepage / landing |
| admission.html | Student admission form |
| fees.html | School fees information |
| admin-login.html | Administrator login |
| admin-dashboard.html | Admin management panel |
